FLASCHENPOST(1)
===============

NAME
----
flaschenpost - a small and simple blog

SYNOPSIS
--------
Flaschenpost is a blog that tries to keep it simple. It uses plain text AsciiDoc
files in a folder as input, and serves them as HTML. You can use whatever way
you want to get your files in that folder - for example, your favorite version
control system.
